#### Hoover App
[Java 8](http://www.oracle.com/technetwork/java/javase/downloads/jre8-downloads-2133155.html) and [Maven](https://maven.apache.org/) should be installed.


##### Building
```mvn compile```

##### Testing
```mvn test```

##### Running
```mvn exec:java -Dexec.mainClass="com.yoti.HooverApp"```
